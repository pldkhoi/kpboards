# TDD Workflow (Mirror)

Source of truth: `.cursor/skills/tdd-workflow/SKILL.md`

## How to apply

1. Write failing tests first.
2. Implement the minimum code to pass.
3. Refactor while keeping tests green.
4. Verify coverage and behavior with Bun-based test commands.
